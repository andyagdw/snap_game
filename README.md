# Snap

The classic _Snap_ game available to play in both single-player and multi-player mode.

## Getting Started

### Prerequisites
* Java JDK 11 or later installed and java/javac on your PATH 
* Git installed
* IntelliJ IDEA (recommended)

#### Using IntelliJ IDEA (recommended)
If IntelliJ IDEA has no project open

1. Click on **clone repository**.
2. Paste either of these URLs and click Clone:
   * https://github.com/andyagdw/snap_game.git
   * git@github.com:andyagdw/snap_game.git

If a project is already open:

1. Go to File ‚Üí New ‚Üí Project from Version Control....
2. Paste one of the URLs above and click **Clone**.

After cloning:

Open the project in IntelliJ IDEA if it does not open automatically.

In the Project view navigate to `src/main/java/org/example` and open `Main.java`.

To run: click the green run icon next to the main method or right-click Main.java ‚Üí Run 'Main.main()'.

#### Run from the command line (alternative)

```
clone (choose https or ssh)
git clone https://github.com/andyagdw/snap_game.git
cd snap_game

# compile
javac -d out src/main/java/org/example/Main.java

# run
java -cp out org.example.Main
```
